# container

> A Nix container for usage with Forgejo Actions

## Components

- [Attic][attic]
- [Bash][bash]
- [busybox][busybox]
- [Cachix][cachix]
- [curl][curl]
- [Nix][nix] with [flakes][flakes] enabled
- [Git][git] with [LFS][lfs]
- [skopeos][skopeo] default container policy

[attic]: https://docs.attic.rs
[bash]: https://gnu.org/software/bash
[busybox]: https://busybox.net/
[cachix]: https://cachix.org
[curl]: https://curl.se
[nix]: https://nixos.org
